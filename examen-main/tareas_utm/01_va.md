# TAREA: EJERCICIO 1 - DOMINIO Y ENTIDADES (REVISADO)

## REQUERIMIENTOS TÉCNICOS:
    * **Capa:** Domain (Entidades Puras).
    * **Objetivo:** Genera las clases de dominio para 'food' usando C# 14. Utiliza la palabra clave 'field' y 'required' para validaciones nativas y seguridad contra nulos:
        * Restaurante: Id (Key), Nombre (required), Especialidad, HorarioApertura, HorarioCierre.
        * Pedido: IdPedido (Key), IdUsuario, FechaHora, CostoEnvio (validar con field >= 0).
        * DetallePedido: IdDetalle (Key), IdPedido, IdProducto, NombreProducto (required), Cantidad (validar con field > 0), Subtotal.
        * Product: Id (Key), Nombre (required), Stock, Precio.
